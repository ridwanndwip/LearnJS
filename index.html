Note for JS tutorial

  • Metode lain yang penting untuk proyek ini adalah metode .pop(). Ini menghapus elemen teraklir dari array dan mengembalikan elemen itu.
    Ketika suatu metode mengembalikan suatu nilai, Anda dapat menganggapnya sebagai mengembalikan nilai tersebut kepada Anda, sehingga tersedia untuk digunakan di bagian lain dari kode Anda.

for (let i = 0; i < count; i = i + 1) {
rows.push(i);
}

Penjelasan : membuat variabel i sampai kurang dari nilai count, lalu memasukkan ke variabel rows(array), jadi sekarang array rows berisi 1 2 3 4 5 6 7 8
